# FruitMarket
--< 中文說明 >--

這是我們這組的水果攤(Fruit Market)，程式碼的說明如下 :

-- 輸入 --
一開始需要使用者輸入水果的種類，然後再輸入水果的數量，而輸入的日期會根據您輸入的現實時間決定，並且會顯示在 Label 裡面。
如果輸入完畢時，可以在命令列提示 "Category (q to quit) :" 時輸入 q 來退出輸入。

-- 輸出 --
會根據現在的水果種類、數量來畫圓餅圖。
如果幫您在輸入時輸入相同的水果種類名稱時，會因為您輸入的時間不同(即使只相差 1 秒)，而分別顯示圓餅圖的不同區域。

--< English Description >--

This is our group’s Fruit Market program. The code works as follows:

-- Input --
At the beginning, the user needs to enter the type of fruit, and then the quantity of that fruit. The input date is determined based on the current real-world time and will be displayed in the label.
After finishing the input, when the prompt in the command line shows "Category (q to quit) :", you can type q to exit the input process.

-- Output --
A pie chart will be drawn based on the current fruit types and quantities.
If you enter the same fruit type at different times (even with a 1-second difference), it will be displayed as separate segments in the pie chart because the input time is different.
